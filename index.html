About the Project

This project was inspired by the need for a centralized platform for musicians to easily access instruments for rent, sale, or repair. As a musician myself, I wanted to create a space that facilitates the exchange of musical instruments and services within the community.

This is also my Portfolio Project for Holberton School, where I was tasked with developing a full-stack application for my final project.
